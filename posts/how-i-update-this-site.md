Two things to remember:

1. **New blog post** — add a `.md` file to `posts/`, commit, push.
2. **New gallery item** — add an image to `public/images/gallery/`, then add a matching entry in `lib/gallery.js`.

Delete this post once you're comfortable with the workflow — it's just a placeholder reminder.
